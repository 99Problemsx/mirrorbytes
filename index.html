Häufig gestellte Fragen

Alles was du über Pokemon Shadows of Legacy wissen musst

Wann wird Pokemon Shadows of Legacy veröffentlicht?

Wir planen die Veröffentlichung für Q3 2024. Aktuell befinden wir uns in der Beta-Phase und arbeiten intensiv an der Story-Integration und dem Balancing.

Wird das Spiel kostenlos sein?

Ja! Pokemon Shadows of Legacy wird komplett kostenlos sein. Wir entwickeln das Spiel aus Leidenschaft für die Pokemon-Community.

Auf welchen Plattformen wird das Spiel verfügbar sein?

Das Spiel wird für Windows PC verfügbar sein. Wir prüfen auch die Möglichkeit einer Mac-Version, können aber noch nichts versprechen.

Wie kann ich Beta-Tester werden?

Tritt unserem Discord-Server bei! Dort werden regelmäßig Beta-Versionen geteilt und du kannst dich für das Testing-Programm anmelden.

Welche Systemanforderungen hat das Spiel?

Minimum: Windows 10, 4GB RAM, 2GB freier Speicherplatz
Empfohlen: Windows 11, 8GB RAM, 4GB freier Speicherplatz

Kann ich das Spiel auf Deutsch spielen?

Ja! Das Spiel wird vollständig auf Deutsch verfügbar sein. Wir arbeiten auch an einer englischen Version.

Gibt es Online-Features oder Multiplayer?

Pokemon Shadows of Legacy ist ein Singleplayer-Erlebnis. Wir konzentrieren uns auf eine tiefgreifende Story und ein ausgewogenes Gameplay.

Wie lange wird das Spiel sein?

Wir planen eine Spielzeit von 15-20 Stunden für die Hauptstory, mit zusätzlichen Inhalten und Sidequests für weitere 10+ Stunden.
